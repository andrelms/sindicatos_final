Sobre o Sistema

O Sistema de Automação de Sindicatos é uma plataforma completa para coleta, processamento e análise de dados relacionados a convenções coletivas e acordos sindicais.

Utilizando tecnologias avançadas de inteligência artificial, como o Gemini 2.5 Pro, o sistema é capaz de extrair informações estruturadas de documentos PDF e Excel, além de monitorar sites e redes sociais de sindicatos para manter a base de dados sempre atualizada.

Todas as informações são rastreáveis, permitindo verificar a fonte original de cada dado apresentado nos painéis e consultas.

Recursos Técnicos
  • Processamento de PDFs e arquivos Excel
  • Integração com Gemini 2.5 Pro
  • Sistema RAG para consultas contextualizadas
  • Automação de coleta de dados
  • Monitoramento de redes sociais
  • Visualização de dados por estado e categoria
